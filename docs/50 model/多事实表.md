# 多事实表

多事实表是指在在一个分析模型中使用多个事实表来支持分析需求的情况。这些事实表可能涉及不同的主题或业务流程，但它们都属于同一个数据仓库。例如，一个企业可能有一个销售事实表，用于跟踪销售额、销售量等指标，同时也有一个库存事实表，用于跟踪库存水平、采购量等指标。

## 如何创建多事实表模型

设置多个表或视图为”事实表“。
