# 时间汇总

我们有“天”粒度的数据，如何汇总到“月”、“季度"或”年“？

## 两个方法：

1. 创建”时间维度“表，选择时间维度中的不同粒度时间字段进行汇总。
2. Datafor提供时间的汇总功能
